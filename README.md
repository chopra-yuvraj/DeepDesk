## Why DeepDesk?
As a **B.Tech CSE student at VIT** and **B.S. Data Science student at IIT Madras**, I wanted to build a modern user interface applying fundamental internet architecture concepts. DeepDesk is an intelligent workspace interface built for distraction-free productivity. 

This project serves as a productivity hub that brings several tools into one single interface so users don't have to constantly switch web browser tabs. It was developed to synthesize modern Web Programming concepts, relying entirely on native JavaScript Objects, Loops, and the core React Dataflow instead of external state libraries.

### Key Features
- **Pomodoro Countdown Timer** - Built with presets and custom controls that calculate time left using JavaScript. 
- **Dynamic Task Manager** - Add and track goals using a dynamic list powered by React State.
- **Experiment Logger** - Log detailed notes and organize them logically using categorized inputs.
- **Architecture Whiteboard** - A digital scratchpad layered over the interface using the HTML5 Canvas API.
- **Vision Board** - An interactive drag-and-drop grid to keep visual inspiration easily accessible.
- **Floating Soundscape** - Immersive ambient audio visualizer featuring Rain, White Noise, and Cyberpunk audio profiles to enhance focus.

---

### Application Interaction
| Feature | Action | Experience |
|---------|--------|------------|
| **Smart Timer** | Start a focus block with the Pomodoro Timer | Updates the React State, which updates the DOM instantly without any page reloads. |
| **Task Manager** | Write down goals | Push new tasks to the list using seamless React Dataflow and local storage persistence. |
| **Architecture Whiteboard** | Sketch out ideas | Draw directly over your workspace using real-time Canvas rendering. |
| **Ambient Audio** | Toggle the Floating Soundscape | Smooth Framer Motion animations reveal a dynamic audio frequency visualizer mapped to the active sound. |

### Engineering Highlights
- **Frontend Architecture**: Runs entirely on the client side using the React Component API.
- **State Management**: Completely relies on React Dataflow (State and Props) and LocalStorage without external backend databases.
- **DOM Manipulation**: Dynamic rendering utilizing React JSX and advanced JavaScript hooks.
- **Advanced Rendering**: Implements Canvas and Drawing tools, utilizing JavaScript Event Listeners to track mouse movement for real-time sketching.
